# Vesper Skills

Use these skills as routed project expertise. Load only the skills needed for the current task.

- `vesper-agent`: full feature-to-PR workflow.
- `vesper-feature-architect`: architecture and planning.
- `vesper-plan-auditor`: plan review.
- `vesper-code-reviewer`: final diff review.
- `vesper-build-engineer`: TypeScript, npm, Vitest, packaging.
- `vesper-safety-engineer`: command safety, approvals, secrets, guardrails.
- `vesper-discord-engineer`: Discord commands, permissions, approvals, threads.
- `vesper-vm-platform-engineer`: UTM, Ubuntu, bootstrap scripts, systemd.
- `vesper-worker-engineer`: Codex runner, queue, worktrees, artifacts.
- `vesper-memory-evals-engineer`: memory, retrospectives, evals.
